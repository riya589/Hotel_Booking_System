# 🏨 Online Hotel Booking System

## 📌 Project Overview

The **Online Hotel Booking System** is a web-based application that allows users to search, view, and book hotel rooms easily. It provides a seamless interface for both users and administrators to manage bookings efficiently.

---

## 🚀 Features

* 🔍 Search hotels and available rooms
* 🛏️ View room details and pricing
* 📅 Book rooms with date selection
* 👤 User registration and login system
* 🧾 Booking confirmation and records
* 🛠️ Admin panel to manage rooms and bookings

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js / Express.js *(if used)*
* **Database:** MongoDB *(if used)*
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/riya589/Hotel_Booking_System.git
```

### 2. Navigate to project folder

```
cd Hotel_Booking_System
```

### 3. Install dependencies

```
npm install
```

### 4. Run the project

```
npm start
```

---

## 💻 Usage

1. Register/Login as a user
2. Browse available hotels and rooms
3. Select dates and book rooms
4. Admin can manage bookings and rooms
