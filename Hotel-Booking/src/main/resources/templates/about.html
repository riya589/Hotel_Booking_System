About This Project

Built for learning. Designed for real use.

HotelHaven is a full-stack hotel booking and management system developed as a college project, demonstrating modern web application architecture from end to end.

What is HotelHaven?

HotelHaven is a production-inspired hotel booking platform where users can browse hotels, filter by city and room type, check date-wise availability, and complete a booking with a full invoice and simulated payment flow. Admins can manage hotels, rooms, and bookings through a dedicated dashboard.

The project was designed to simulate a real-world software system — with JWT-based authentication, role-based access control, clean REST APIs, and a Thymeleaf-powered frontend — while remaining approachable for academic demonstration and evaluation.

Two Roles

👤 Customer
Browse hotels, book rooms, view invoices, manage bookings
🛡️ Admin
Add & edit hotels and rooms, manage all bookings and statuses

Key Features

  • ✅ JWT-secured login & registration
  • ✅ Role-based access (Customer vs Admin)
  • ✅ Hotel search with city, price & room type filters
  • ✅ Date-aware room availability (no double-booking)
  • ✅ Full booking flow with confirmation screen
  • ✅ Invoice generation per booking
  • ✅ Simulated payment processing
  • ✅ Admin dashboard with stats and management tools
  • ✅ Hotel image gallery support
  • ✅ Mobile-responsive design
